# NeoArch Documentation

Welcome to the NeoArch Linux documentation.

NeoArch is an Arch and Artix based rolling-release distribution that combines Arch's package ecosystem with init system freedom and intelligent package management via NAPM.

- [General](/general) - Getting started, installation guide, and general system information
- [NAPM](/napm) - The NeoArch Package Manager: commands, cache, and auto-repair
